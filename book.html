Q - What is HTML?
A - HTML is Hyper Text Markup Language.

Q - What is Hyper Text Markup Language?
A - Hyper Text Markup Language is the Standard Markup Language used for creating and designing web pages. It defines the structure and layout of a web document by using a varity of tags and attributes.

Q - What is Tags and Attributes?
A - In HTML, tags and attributes are fundamental concepts used to structure and define elements within a web page. HTML tags define the structure and content of elements. While attributes provide additional information or modify the behavior and appearance of those elements. Together, they form the building blocks of HTML documents, allowing web developers to create structured and interactive web pages.
